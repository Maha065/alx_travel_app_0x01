# alx_travel_app_0x01

This project provides API endpoints for managing **Listings** and **Bookings**.

## Endpoints

- `/api/listings/` → CRUD for listings
- `/api/bookings/` → CRUD for bookings
- `/swagger/` → Swagger API documentation
- `/redoc/` → Redoc API documentation
